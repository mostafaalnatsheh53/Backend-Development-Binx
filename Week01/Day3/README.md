# Day 3 – Object-Oriented Programming (OOP)

## Overview
In this project, I practiced the fundamental concepts of Object-Oriented Programming (OOP) in C# by building a simple Library System.

## Topics Covered
- Classes
- Records
- Structs
- Encapsulation
- Constructors
- Inheritance
- Interfaces
- Polymorphism

## Project
The project includes:
- `Book` class
- `Member` class
- `BorrowRequest` record
- `IPrintable` interface
- A polymorphic method that prints information for any object implementing `IPrintable`

## OOP Concepts Used
- Encapsulation using private fields and public properties
- Constructor validation
- Interface implementation
- Polymorphism through interfaces

## What I Learned
- How to design classes and records
- The difference between classes, records, and structs
- How inheritance and interfaces work
- How polymorphism makes code more flexible and reusable
- How to apply OOP principles in a small domain model

## Technologies
- C#
- .NET
- Visual Studio Code
